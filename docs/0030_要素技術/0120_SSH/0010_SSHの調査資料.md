# SSH接続できない場合の対象方法
### OSを再インストールした場合など
* SSH接続先のホスト名の古いSSHキーがSSH接続元PC側に残っているとエラーとなる
* SSH接続元PCの/{user}/.ssh/known_hostsファイル中のSSH接続先のホスト名の古いSSHキーを削除する
